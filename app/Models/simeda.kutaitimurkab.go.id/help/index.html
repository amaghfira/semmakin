Alamat Website

  • Website diakses pada alamat : s.bps.go.id/omae
  • Saat ini masih di server BPS Prov Jateng
  • Ada opsi untuk dipindahkan ke server Diskominfo Prov Jateng

Role & Hak Akses

  • Pengelolaan user/instansi melalui menu Report -> User
  • OPD :

    Entri MS Kegiatan, MS Variabel, serta MS Indikator

  • BPS :

    Verifikasi data, pengelolaan user/instansi

  • Diskominfo :

    Approval selaku walidata

Pengelolaan User/Instansi

  • User di Kab/Kota dikelola oleh masing-masing BPS Kab/Kota
  • Tata cara pengisian
    • Role BPS menggunakan username : bps33xx
    • Role OPD dan Diskominfo username diawali kodewilayah_ : 3301_diskominfo 3301_pertanian
    • Secara default, password=username
  • Field yang wajib diisi : nama instansi, role, wilayah, username

Urutan Proses

  1. OPD melakukan entri MS Kegiatan, MS Variabel, serta MS Indikator
  2. BPS melakukan verifikasi data, memberikan feedback
  3. OPD memeriksa hasil validasi, memperbaiki input data, memberikan respon
  4. Diskominfo memberikan approval kegiatan (termasuk variabel dan indikator di dalamnya)
